# Tarea de simulación - Yahtzee con Método Montecarlo

# Descripción
Este proyecto implementa una simulación del juego Yahtzee para dos jugadores utilizando el método de Montecarlo para la toma de decisiones.
El sistema simula múltiples escenarios posibles en cada turno para determinar qué dados conservar y maximizar la puntuación.

# Objetivo
Aplicar el método de Montecarlo en un problema basado en probabilidad y analizar el comportamiento del sistema mediante simulaciones.

# Tecnologías utilizadas
Python 3.x
Librerías:
random
collections

# Método Montecarlo aplicado
Simulación de lanzamientos:
Generación de números aleatorios con distribución uniforme.
Toma de decisiones:
Simulación de múltiples combinaciones de dados para elegir la mejor opción.
Análisis estadístico:
Ejecución de múltiples partidas para calcular métricas como promedio y desviación estándar.


# Funcionalidades
1 Juego completo de Yahtzee (13 turnos por jugador)
2 Estrategia basada en simulación Montecarlo
3 Análisis estadístico de múltiples partidas
4 Ingreso de nombres de jugadores
5 Uso opcional de semilla para reproducibilidad

# Ejemplo de resultados
Jugador 1: 171 puntos
Jugador 2: 136 puntos
Promedios en simulación:
J1: 122.18
J2: 127.52

Conclusiones
El método de Montecarlo permite mejorar la toma de decisiones.
El juego presenta alta variabilidad debido al azar.
La simulación permite analizar el comportamiento probabilístico del sistema.
